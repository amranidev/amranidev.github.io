Scaffold-interface

Welcome to the scaffold-interface documentation. This is a work in progress and will be continuously updated. Pull requests are welcome so if you would like to contribute, please fork me on GitHub.

What is scaffold-interface

Scaffold-interface is a smart CRUD generator for Laravel that generate your models,views,controllers and One-To-Many relationships just in few clicks. with elegant design and template specification for views.

Requirements

  • php >= 5.5.0

Getting Started

Installation

  1. Add scaffold-interface to your composer.json file to require Scaffold-Interface :

    composer require Amranidev/scaffold-interface:1.3.*

  2. Update Composer :

    composer update

  3. Add the service providers to config/app.php :

    Amranidev\ScaffoldInterface\ScaffoldInterfaceServiceProvider::class, Amranidev\Ajaxis\AjaxisServiceProvider::class,

  4. Publish the assets in your application with :

    $ php artisan vendor:publish

  5. Migrate scaffoldinterface :

    $ php artisan migrate

    Congratulations, you have successfully installed Scaffold Interface!

Usage

The scaffold-interface have a lot features that save a tons of time for you and for your projects development life-cycle.

Scaffolding

  1. Access to scaffold interface :

    http://{your-project}/scaffold to get into scaffoldinterface.

  2. Table creation :

    Create your table, you can specify many of attributes such like (String,date,longtext,etc.)

  3. After creation :

    To complete your scaffolding . go to your terminal and type.

    $ php artisan migrate

  4. Finally :

    Scaffolding it’s done. go to http://{your-project}/{your-model}
